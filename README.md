# Proffy, plataforma de cadastro e procura de professores onLine
## 2ª Next Level Week da Rocketseat www.rocketseat.com.br


### Instalação:
Baixar o repositório na pasta desejada, abrir o terminal e rodar:

> npm install

### Dependências diretas:
* express 4.17.1
* nunjucks 3.2.2
* sqlite-async 1.1.0
